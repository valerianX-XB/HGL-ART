# OOH Inventory Methodology

The expanded OOH inventory uses the QA-passed corrected OOH candidate dataset and converts it into a public-safe asset schema.

Status taxonomy:
- Confirmed available
- Network bookable
- Potentially bookable
- Occupied / active advertiser observed
- Permitted sign / availability unknown
- Candidate to verify
- Not recommended

No asset is represented as vacant or guaranteed purchasable. Availability, pricing, impressions, and exact unit selection must be confirmed with the vendor before purchase.
