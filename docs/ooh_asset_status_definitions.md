# OOH Asset Status Definitions

- Confirmed available: only if a vendor or marketplace confirms current availability.
- Network bookable: network can be bought publicly, but exact unit/date requires confirmation.
- Potentially bookable: the format is sold by the vendor, but exact unit/date is not confirmed.
- Occupied / active advertiser observed: current/recent advertiser observed; does not mean unavailable forever.
- Permitted sign / availability unknown: permit or sign record exists, no availability evidence.
- Candidate to verify: promising site or format needing vendor/source verification.
- Not recommended: weak fit or high uncertainty.
