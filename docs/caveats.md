# Caveats

HGL ART is treated in this project as a multilingual early childhood education / preschool-style school integrating English, Mandarin Chinese, Spanish, and art-based education.

本地图仅使用公开聚合数据和建模估算，不识别、不追踪、不推断任何个人儿童、家长、照护者、家庭、住户或居民信息。

- All demographic data is aggregate.
- Building-level values are modeled estimates, not actual residents or actual children.
- Street-level values are modeled opportunity signals, not exact street population counts.
- Fine-grain heatmaps are directional market signals, not household-level maps.
- OOH availability, pricing, impressions and exact unit selection must be confirmed by vendors.
- HGL legal/license status must be confirmed before using licensed preschool, licensed daycare, or official kindergarten.
- HGL address and “near Union Square” must be confirmed before live creative.
- Mandarin, Spanish and multilingual claims must match actual staffing and curriculum.
- ZIP/ZCTA 3D extrusion is a low-height visual summary only. It is not a precise street-level view.
- 3D 高度仅用于相对比较，不代表真实建筑高度。
- 建筑高度已为可读性进行视觉压缩；数据抬高仅为符号化表达，不代表真实建筑高度。
